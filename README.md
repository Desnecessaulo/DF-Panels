## Features

- **Boss Timer** - Countdown to next boss spawn + Devil Hound alert
- **Auto Bank** - Quick withdraw/deposit buttons
- **Quick Service** - One-click food/medical/armor repair
- **Quick Buy** - Rapid purchase of common items (Right-click to favorite items)

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) (Chrome/Edge) or [Greasemonkey](https://www.greasespot.net/) (Firefox)
2. Click here to install: https://greasyfork.org/en/scripts/566610-df-panels-beta

## Security

- Uses Dead Frontier's built-in hash() function for all requests
- No credentials stored
- Only external API: dfprofiler.com (read-only boss data)
- Full source code is visible and auditable
